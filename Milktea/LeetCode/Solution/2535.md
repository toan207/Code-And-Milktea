> https://leetcode.com/problems/increment-submatrices-by-one/description

## Bài toán

```
You are given a positive integer n, indicating that we initially have an n x n 0-indexed integer matrix mat filled with zeroes. 

You are also given a 2D integer array query. For each query[i] = [row1i, col1i, row2i, col2i], you should do the following operation: 

Add 1 to every element in the submatrix with the top left corner (row1i, col1i) and the bottom right corner (row2i, col2i). 

That is, add 1 to mat[x][y] for all row1i <= x <= row2i and col1i <= y <= col2i. Return the matrix mat after performing every query.
```

## Điều kiện
- 1 <= n <= 500
- 1 <= queries.length <= 10^4
- 0 <= row1[i] <= row2[i] < n
- 0 <= col1[i] <= col2[i] < n

## Tóm tắt

- Có ma trận **n x n**, ban đầu toàn số `0`.
- Có danh sách query, mỗi query dạng: `[row1, col1, row2, col2]`.
- Với mỗi query, ta cộng 1 vào toàn bộ ô trong hình chữ nhật con có:
  - Góc trên-trái: `(row1, col1)`
  - Góc dưới-phải: `(row2, col2)`
- Ta cộng 1 cho mọi ô trong hình chữ nhật con từ `(r1, c1)` đến `(r2, c2)`.
- Sau khi xử lý tất cả query, trả về **ma trận cuối cùng**.

## Nhận xét

- Cập nhật từng ô trong mỗi query → `O(q * n^2)` → quá chậm.
- Dùng **kỹ thuật đánh dấu biên**:
```
Chỉ cập nhật 4 điểm biên mỗi query [r1,c1,r2,c2]:
mx[r1][c1] +=1
mx[r1][c2 + 1] -=1
mx[r2 + 1][c1] -=1
mx[r2 + 1][c2 + 1] +=1
```
  - `+1` tại góc **trên-trái** -> bắt đầu tăng từ đây.
  - `-1` ở **bên phải** và **phía dưới** -> chặn vùng tăng.
  - `+1` tại góc **dưới-phải+1** -> cân bằng.
- Dùng **prefix sum 2D** lan giá trị từ các điểm biên -> ma trận cuối cùng.
- Độ phức tạp: O(n^2), tối ưu cho n <= 10^4.

## Mã giả

```
mx <- n x n zero matrix

for each [r1, c1, r2, c2] in queries:
    mx[r1][c1] += 1
    if c2+1 < n: mx[r1][c2+1] -= 1
    if r2+1 < n:
        mx[r2+1][c1] -= 1
        if c2+1 < n: mx[r2+1][c2+1] += 1

for i from 0 to n-1:
    for j from 1 to n-1:
        mx[i][j] += mx[i][j-1]

for j from 0 to n-1:
    for i from 1 to n-1:
        mx[i][j] += mx[i-1][j]

return mx
```
