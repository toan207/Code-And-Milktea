> https://leetcode.com/problems/set-intersection-size-at-least-two/description

## Bài toán

```
You are given a 2D integer array intervals where intervals[i] = [starti, endi] represents all the integers from starti to endi inclusively. 

A containing set is an array nums where each interval from intervals has at least two integers in nums. 
For example, if intervals = [[1,3], [3,7], [8,9]], then [1,2,4,7,8,9] and [2,3,4,8,9] are containing sets. 

Return the minimum possible size of a containing set.
```

## Điều kiện
- 1 <= intervals.length <= 3000
- intervals[i].length == 2
- 0 <= start[i] < end[i] <= 10^8

## Tóm tắt

- Bạn được cho `intervals[i] = [start, end]`.
- Cần chọn một tập số (set) `nums` sao cho:
  - Mỗi interval phải chọn ít nhất 2 số đưa vào `nums`.
- Trả về kích thước nhỏ nhất có thể của `nums` sau khi loại các số trùng lặp.

## Nhận xét

- Đây là bài **tham lam (Greedy)**.
- Sắp xếp các interval theo:
    - `end` giảm dần.
    - `start` tăng dần.
- Duyệt qua các interval:
    - Nếu `end` của interval hiện tại lớn hơn `end` của interval trước đó, chọn 2 số lớn nhất trong interval hiện tại để thêm vào `nums`.
    - Nếu `end` của interval hiện tại nhỏ hơn hoặc bằng `end` của interval trước đó và `start` của interval hiện tại lớn hơn `start` của interval trước đó, chọn 1 số lớn nhất trong interval hiện tại để thêm vào `nums`.
    - Nếu không trong cả 2 trường hợp trên, không cần chọn số nào.
- Vì luôn chọn số càng lớn càng tốt -> tối ưu cho các interval tiếp theo.

## Mã giả

```
sort intervals by (end asc, start desc)

setSize = 0
small, big = -1, -1

for each interval in intervals:
    if interval.end > big:
        setSize += 2
        small = interval.end - 1
        big = interval.end

    else if interval.start > small:
        setSize += 1
        small = big
        big = interval.end

return setSize
```
