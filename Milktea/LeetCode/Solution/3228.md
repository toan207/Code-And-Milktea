## Bài toán

```
You are given a binary string s. 
You can perform the following operation any number of times:
- Choose any index i (i + 1 < s.length) such that s[i] == '1' and s[i + 1] == '0'.
- Move the character s[i] to the right until it reaches the end of the string or another '1'.

For example:
s = "010010" choose i = 1 → s becomes "000110".

Return the maximum number of operations you can perform.
```

## Điều kiện
- 1 <= s.length <= 10^5
- s[i] ∈ {'0','1'}

## Tóm tắt

- Chuỗi **s** gồm các ký tự `'0'` và `'1'`.
- Mỗi lần chọn `'1'`, ta di chuyển `'1'` sang phải đến khi gặp kí tự '1' tiếp theo hoặc hết chuỗi.
- Cần tính tối đa số thao tác có thể thực hiện.

## Nhận xét

- Nếu duyệt từng bước chuỗi này sẽ tốn thời gian **O(n²)** -> vượt quá thời gian với 10^5.
- Ta nhận ra rằng mỗi kí tự `'1'` có thể **“vượt qua”** mọi kí tự `'0'` đứng sau nó (cho đến khi gặp kí tự `'1'` khác).
- Thay vì duyệt tuần tự, ta chỉ cần đếm tổng số cặp `'1'` trước `'0'` có thể di chuyển.

## Mã giả

```
res <- 0
ones <- 0
flag <- 1

for i from 0 to len(s)-1:
    if s[i] == '1':
        ones <- ones + 1
        flag <- 1
    else:
        res <- res + ones * flag
        flag <- 0
return res
```
